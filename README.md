# 🎨 The Frame Gallery

A responsive sidebar navigation menu built using pure HTML and CSS.

This project was created to strengthen my frontend fundamentals by implementing a clean, modern sidebar UI featuring smooth animations and a glassmorphism design.

## ✨ Features

- Responsive sidebar navigation
- Pure HTML & CSS (No JavaScript)
- Modern Glassmorphism UI
- Smooth hover effects
- Fixed sidebar layout
- Font Awesome icons
- Mobile-friendly design
- Clean and organized code

---

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome
- Google Fonts
